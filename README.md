# 🎓 Django School Management System

A dynamic web application built using Django for managing school-related data such as students, courses, contact details, and website content.

---

## 🚀 Features

* Dynamic Website Pages (Home, About, Contact)
* Admin Panel for Content Management
* Contact Form with Database Storage
* Image Upload (Slider, Items, Testimonials)
* Reusable Templates (Header, Footer)
* Clean and Structured Django Project

---

## 🛠️ Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, Bootstrap
* Database: SQLite
* Version Control: Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/bhatnagardip-spec/django-school-project.git
```

2. Navigate to project folder:

```
cd django-school-project
```

3. Create virtual environment:

```
python -m venv myenv
```

4. Activate environment:

```
myenv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

6. Run migrations:

```
python manage.py migrate
```

7. Start server:

```
python manage.py runserver
```
